# Contributing to FTCoE Computational Framework

We welcome contributions from physicists, neuroscientists, mathematicians, and programmers!

## Ways to Contribute

- 💻 **Code**: Add new simulations or improve existing ones
- 📐 **Mathematics**: Help formalize equations
- 🔬 **Science**: Design testable predictions
- 📝 **Docs**: Improve documentation
- 🐛 **Testing**: Find and report bugs

## Getting Started

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a Pull Request

## Code Style

- Use descriptive variable names
- Include docstrings for all functions
- Add comments for complex algorithms
- Follow PEP 8 guidelines

## Questions?

Open an issue or contact the Architect.
